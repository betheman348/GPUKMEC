# GP Tools

A small collection of free, browser-based educational references for UK general practice. Built by Dr BN, a UK GP, for UK clinicians.

Live at: [gp-tools.uk](https://gp-tools.uk)

## What this is

An educational reference site that surfaces published clinical guidance in a faster, filterable way than the source PDFs. Currently hosts:

- **UKMEC 2025 Reference** — interactive view of the CoSRH UK Medical Eligibility Criteria for contraceptive use, filtered by scenario factors.

## What this is not

- Not a medical device under MHRA classification.
- Not a clinical decision-support system that makes patient-specific assessments.
- Not a substitute for the full published guidance.
- Not a replacement for clinical judgement, training, or professional indemnity.

All clinical decisions remain with the qualified healthcare professional. The tools surface what published guidance says; the user reads, interprets, and applies it.

## Stack

- Static HTML / CSS / vanilla JavaScript.
- Hosted on Cloudflare Pages, deployed from GitHub.
- No backend. No database. No cookies. No analytics. No tracking.
- Everything runs in the user's browser.
